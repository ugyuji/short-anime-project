# コンセプト開発ドキュメント

> **プロジェクト名**: フラッグオリジナルショートアニメプロジェクト
> **作成日**: 2026-01-16
> **フェーズ**: 2. コンセプト開発

---

## 概要

フェーズ1（戦略設計）で選定した2つのテーマについて、キーワードブレインストーミングからモチーフ候補の導出までを行った。

| テーマ | 分類 |
|--------|------|
| 「日常のタイミングが悪いこと」 | 共感型 |
| 「小さな復讐が成功する」 | 爽快型 |

---

## テーマ1: 「日常のタイミングが悪いこと」

### 視聴者への約束

> 「わかる！」という共感と、諦めの笑い

### 1.1 キーワードブレインストーミング

| カテゴリ | キーワード |
|---------|-----------|
| **感情** | あきらめ、脱力、やるせなさ、苦笑い、「知ってた」、ため息、無表情、虚無 |
| **状況** | 傘と雨、電車、信号、レジ、トイレ、充電、エレベーター、既読、写真 |
| **性格** | 不運体質、ポジティブ、めげない、鈍感、天然、お人好し、真面目 |
| **行動** | 走る→間に合わない、待つ→来ない、避ける→当たる、準備→無駄 |
| **見た目** | ゆるい、脱力系、丸い、ズレてる、小さい、シンプル |
| **世界観** | 日常、現代、都会、通勤、オフィス、学校、コンビニ |
| **関係性** | 一人、孤軍奮闘、周りは幸運、対比、見守られてる |

### 1.2 キーワード絞り込み（10個）

| # | キーワード | 選定理由 |
|---|-----------|---------|
| 1 | **あきらめ顔** | コア表現。共感の入口 |
| 2 | **不運体質** | キャラクターの宿命 |
| 3 | **めげない** | 愛される理由（隙） |
| 4 | **タイミングズレ** | テーマの核心 |
| 5 | **日常** | 誰もが経験する世界 |
| 6 | **脱力系** | ビジュアルの方向性 |
| 7 | **丸い** | 可愛さ、親しみ |
| 8 | **一人** | 孤軍奮闘感 |
| 9 | **苦笑い** | リアクションの基本 |
| 10 | **「知ってた」** | 決め台詞候補 |

### 1.3 モチーフ候補

| # | 掛け合わせ | モチーフ | 理由 |
|---|-----------|---------|------|
| A | 不運 × 丸い × めげない | **ハムスター** | 回し車で頑張っても進まない |
| B | あきらめ × 脱力 × 日常 | **猫** | 「まあいいか」感、日常感 |
| C | タイミング × 一人 × 苦笑い | **時計の精** | タイミングの擬人化 |
| D | 不運 × 丸い × 小さい | **だるま** | 転んでも起き上がる |
| E | ズレてる × めげない × 脱力 | **スライム系** | 形が定まらない＝ズレてる |

### 1.4 シチュエーション例

| シーン | 展開 |
|--------|------|
| 傘 | 傘を持ってきた日に限って晴れる。持ってこなかった日に限って豪雨 |
| 電車 | 走って乗った電車が「次の駅で5分停車します」 |
| レジ | 自分の番になった瞬間「レジ交代します」 |
| 信号 | 急いでる時だけ全部赤信号 |
| 充電 | 出かける直前にスマホ充電3% |
| エレベーター | 待ってたら来ない。諦めて階段行ったら来る |

## テーマ2: 「小さな復讐が成功する」

### 視聴者への約束

> 日常のイラッとを代わりに晴らす、スカッとする体験

### 2.1 キーワードブレインストーミング

| カテゴリ | キーワード |
|---------|-----------|
| **感情** | スカッと、痛快、ざまあみろ、してやったり、ニヤリ、爽快、達成感 |
| **状況** | 割り込み、横取り、マウント、無視、理不尽、パワハラ、迷惑行為 |
| **性格** | 普段は大人しい、キレたら怖い、策士、天然で復讐、無自覚 |
| **行動** | やり返す、仕返し、天罰、因果応報、ブーメラン、自爆誘導 |
| **見た目** | 可愛いのに黒い、ギャップ、笑顔が怖い、無表情 |
| **世界観** | オフィス、電車、SNS、日常のイラッと場面 |
| **関係性** | 加害者と被害者、第三者視点、神視点 |

### 2.2 キーワード絞り込み（10個）

| # | キーワード | 選定理由 |
|---|-----------|---------|
| 1 | **スカッと** | コア感情 |
| 2 | **因果応報** | ストーリーの型 |
| 3 | **普段は大人しい** | ギャップの源泉 |
| 4 | **無自覚** | 天然復讐の面白さ |
| 5 | **可愛いのに黒い** | ビジュアルギャップ |
| 6 | **日常のイラッと** | 共感できるネタ元 |
| 7 | **ニヤリ** | 決め表情 |
| 8 | **ブーメラン** | 復讐のパターン |
| 9 | **天罰** | 自分の手を汚さない |
| 10 | **してやったり** | 決め台詞候補 |

### 2.3 モチーフ候補

| # | 掛け合わせ | モチーフ | 理由 |
|---|-----------|---------|------|
| A | 可愛い × 黒い × ニヤリ | **黒猫** | 不吉の象徴、可愛い |
| B | 天罰 × 無自覚 × 天然 | **天使（堕天使）** | 天罰を下す存在 |
| C | 因果応報 × ブーメラン | **鏡の精** | 行いを跳ね返す |
| D | 大人しい × ギャップ × スカッと | **うさぎ** | 普段可愛い→キレると怖い |
| E | 日常 × 復讐 × 無表情 | **人形/マスコット** | 無表情で淡々と復讐 |
| F | 因果応報 × ブーメラン × スカッと | **ブーメラン** | 投げたものが返ってくる＝因果応報の象徴 |

### 2.4 シチュエーション例

| シーン | 展開 |
|--------|------|
| 割り込み | 電車で割り込んできた人が、降りる駅でドアが開かない側に |
| マウント | SNSでマウントしてきた人のアカウントが凍結 |
| 横取り | 横取りした席が実は濡れてた |
| 無視 | 挨拶を無視した人が、その日ずっと運が悪い |
| パワハラ上司 | 理不尽な指示をした上司が、自分で同じ目に遭う |
