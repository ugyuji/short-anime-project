# キャラクター設計ドキュメント

> **プロジェクト名**: フラッグオリジナルショートアニメプロジェクト
> **作成日**: 2026-01-16
> **フェーズ**: 3. キャラクター設計

---

## 概要

フェーズ2（コンセプト開発）で選定したモチーフに基づき、2つのキャラクターの詳細設計を行った。

| テーマ | モチーフ |
|--------|---------|
| 「日常のタイミングが悪いこと」 | スライム系 |
| 「小さな復讐が成功する」 | ブーメラン |

---

## キャラクター1: スライム系

### テーマ

> 「日常のタイミングが悪いこと」（共感型）

### 視聴者への約束

> 「わかる！」という共感と、諦めの笑い

---

### 1.1 世界観

| 項目 | 設定 |
|------|------|
| **舞台** | 現代日本の都会（駅、オフィス、コンビニ等） |
| **時代** | 現代 |
| **設定** | 人間社会にひっそり暮らすスライム。人間と同じ生活をしようとするが、タイミングがことごとく悪い |
| **ルール** | スライムなので形が不安定＝常にズレてる。急ぐと形が崩れる |
| **出生地** | YouTube Shorts |
| **生息地** | SNS、グッズ展開 |

---

### 1.2 プロフィール

#### 基本情報

| 項目 | 設定 |
|------|------|
| **名前** | （候補）ズレたろう / タイミー / ヌルッと |
| **年齢** | 不明（見た目は若い社会人風） |
| **性別** | 無性別 |
| **誕生日** | 4月1日（エイプリルフール＝ズレの象徴） |
| **身長** | 伸び縮みする（基本30cm） |

#### 性格・内面

| 項目 | 設定 |
|------|------|
| **性格（3語）** | めげない、鈍感、前向き |
| **口癖** | 「...だと思った」「まあいっか」 |
| **好きなもの** | 予定通りにいくこと（叶わない） |
| **嫌いなもの** | 特になし（鈍感なので気づかない） |
| **特技** | どこでも入り込める（形が変わる） |
| **弱点（隙）** | タイミングが致命的に悪い |
| **夢** | 一度でいいからタイミングよく行動したい |

---

### 1.3 「隙」の設計

| パターン | 設定 | 効果 |
|---------|------|------|
| **能力的限界** | タイミングが絶望的に悪い | 視聴者の共感、応援したくなる |
| **性格的欠点** | 鈍感で気づかない | 天然ボケ感 |
| **外見的特徴** | 形が安定しない | ズレてる感を視覚化 |
| **ギャップ** | 頑張ってるのに報われない | 愛おしさ |

---

### 1.4 行動原理とリアクション

| シチュエーション | リアクション |
|----------------|-------------|
| 嬉しいとき | ぷるぷる震える（でもすぐ不運が来る） |
| 悲しいとき | 少し溶ける、でもすぐ復活 |
| 困ったとき | 「...だと思った」と諦め顔 |
| 驚いたとき | びよーんと伸びる |
| 成功したとき | ほぼない（あっても次の瞬間台無し） |

#### 禁止行動（NGリスト）

- 誰かを恨む、怒る
- 不運を人のせいにする
- 諦めて動かなくなる

---

### 1.5 コンテンツ構成パターン

#### パターンA: 淡々と不運が続く

```
[0:00] 朝、目覚ましが鳴る5分前に起きる
[0:05] 急いで準備
[0:10] 電車に駆け込む
[0:15] 「この電車は回送になります」
[0:20] 諦めの表情
[0:25] テロップ「...だと思った」
```

#### パターンB: 期待→裏切り

```
[0:00] 「今日こそは！」と意気込む
[0:10] 順調に進む（視聴者も期待）
[0:20] 最後の最後で失敗
[0:25] 「知ってた」の表情
```

## キャラクター2: ブーメラン

### テーマ

> 「小さな復讐が成功する」（爽快型）

### 視聴者への約束

> 日常のイラッとを代わりに晴らす、スカッとする体験

---

### 2.1 世界観

| 項目 | 設定 |
|------|------|
| **舞台** | 現代日本の日常（電車、オフィス、SNS等） |
| **時代** | 現代 |
| **設定** | 因果応報を司る存在。悪い行いをした人に「返ってくる」役割 |
| **ルール** | 悪意のある行動を見つけると、同じことが本人に返る |
| **出生地** | YouTube Shorts |
| **生息地** | SNS、グッズ展開 |

---

### 2.2 プロフィール

#### 基本情報

| 項目 | 設定 |
|------|------|
| **名前** | （候補）ブーメランくん / カエルン / メグリン |
| **年齢** | 不明（永遠の存在） |
| **性別** | 無性別 |
| **誕生日** | 未定 |
| **身長** | 20cm程度（手のひらサイズ） |

#### 性格・内面

| 項目 | 設定 |
|------|------|
| **性格（3語）** | 無表情、淡々、無自覚 |
| **口癖** | 「...」（無言）または「返ってきた」 |
| **好きなもの** | 因果応報が成立する瞬間 |
| **嫌いなもの** | 特になし（感情がない） |
| **特技** | 悪意を感知する、投げられると必ず戻ってくる |
| **弱点（隙）** | 善意の行動には反応できない |
| **夢** | なし（ただ役割を果たすだけ） |

---

### 2.3 「隙」の設計

| パターン | 設定 | 効果 |
|---------|------|------|
| **性格的欠点** | 無感情、無自覚 | シュールさ |
| **外見的特徴** | 可愛いのに黒い目 | ギャップ、不気味可愛い |
| **能力的限界** | 善意には反応できない | 一方的な存在ではない |
| **ギャップ** | 可愛い見た目で淡々と復讐 | 面白さの源泉 |

---

### 2.4 行動原理とリアクション

| シチュエーション | リアクション |
|----------------|-------------|
| 悪意を見つけたとき | 無表情でスッと近づく |
| 復讐が成功したとき | ニヤリ（一瞬だけ） |
| 善意を見たとき | 何もしない、スルー |
| 投げられたとき | 必ず戻ってくる |
| 困ったとき | 困らない（感情がない） |

#### 禁止行動（NGリスト）

- 無関係な人を巻き込む
- 過剰な復讐（あくまで「同じこと」が返るだけ）
- 感情的になる

---

### 2.5 コンテンツ構成パターン

#### パターンA: 因果応報型

```
[0:00] 悪意ある行動（割り込み、マウント等）
[0:10] ブーメランがスッと現れる
[0:15] 無表情で見つめる
[0:20] 同じことが本人に返ってくる
[0:25] ニヤリ → 「返ってきた」
```

#### パターンB: 天罰型

```
[0:00] 理不尽な行動をする人
[0:10] 被害者が我慢している
[0:15] ブーメランが背後に現れる（被害者は気づかない）
[0:20] 加害者に天罰が下る
[0:25] 被害者「？」 ブーメラン（無言で去る）
```
