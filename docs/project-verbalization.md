## 1. テーマ（視聴者価値）

### 1.1 視聴者はなぜ見たいのか

成功しているショートアニメの分析から、視聴者が求める価値は4つに分類される：

| 分類 | 視聴者の欲求 | 代表例 |
|------|-------------|--------|
| **共感型** | 自分を肯定してほしい | 可哀想に！ |
| **中毒型** | 頭を空っぽにしたい | 松尾です、マオメエ |
| **爽快型** | スカッとしたい | そろ谷 |
| **ギャップ型** | おしゃれに笑いたい | マリマリマリー |

### 1.2 テーマ候補

#### 共感型（自分を肯定してほしい）

| # | テーマ | 視聴者への約束 |
|---|--------|---------------|
| 1 | 「報われない毎日を、ゆるく肯定する」 | 頑張っても報われない日常を、可愛く代弁する |
| 2 | 「日常のタイミングが悪いこと」 | 「わかる！」という共感と諦めの笑い |
| 3 | 「小さな失敗を、大げさに悲しむ」 | どうでもいいことで落ち込む自分を肯定 |

#### 中毒型（頭を空っぽにしたい）

| # | テーマ | 視聴者への約束 |
|---|--------|---------------|
| 1 | 「30秒で脳をリセットする」 | 意味不明だけど、なぜか見てしまう中毒性 |
| 2 | 「謎のルーティンを淡々とこなす」 | 意味のない行動を真剣にやる不条理さ |
| 3 | 「会話が全く噛み合わない」 | ズレた掛け合いの心地よさ |

#### 爽快型（スカッとしたい）

| # | テーマ | 視聴者への約束 |
|---|--------|---------------|
| 1 | 「あるあるを、斜め上から切り取る」 | 共感からの予想外で笑わせる |
| 2 | 「小さな復讐が成功する」 | 日常のイラッとを代わりに晴らす |
| 3 | 「面倒なことを一発で解決する」 | やりたくてもできないことを代行 |

#### ギャップ型（おしゃれに笑いたい）

| # | テーマ | 視聴者への約束 |
|---|--------|---------------|
| 1 | 「おしゃれなのに、くだらない」 | ビジュアルと内容の落差で笑わせる |
| 2 | 「真剣な顔で、どうでもいいことを語る」 | シリアスな演出×くだらない内容 |
| 3 | 「エモい雰囲気で、日常のグチ」 | 映える映像で愚痴を美しく昇華 |

### 1.3 選定基準

1. **AI制作との相性**: ビジュアル重視はAI生成の強みを活かせる
2. **差別化可能性**: 既存100万人超チャンネルとの棲み分け
3. **グッズ展開**: キャラクター物販への発展性
4. **継続性**: 週2-3本の更新を維持できるネタの広がり

## 2. ターゲット

### 2.1 コアターゲット

| 項目 | 定義 |
|------|------|
| **年齢層** | 18〜34歳 |
| **行動特性** | YouTube Shortsを日常的に視聴 |
| **心理状態** | 日常のストレスから解放されたい |
| **視聴シーン** | 通勤中、休憩時間、就寝前 |
| **求めるもの** | 短時間で完結する満足感、共感、笑い |

### 2.2 ターゲットインサイト（分類別）

| 分類 | ターゲットの深層心理 |
|------|-------------------|
| 共感型 | 「頑張っても報われない。でも、それを笑いに変えてくれる存在がほしい」 |
| 中毒型 | 「考えたくない。何も考えずに笑いたい」 |
| 爽快型 | 「あるある！と思った瞬間、予想外の展開で裏切られたい」 |
| ギャップ型 | 「ダサいものは見たくない。でもくだらないもので笑いたい」 |
