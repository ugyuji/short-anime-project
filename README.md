# フラッグオリジナルショートアニメプロジェクト

YouTube Shorts向けオリジナルキャラクターアニメーションの企画・制作プロジェクト

---

## プロジェクト概要

| 項目 | 内容 |
|------|------|
| **目標** | チャンネル登録者 3,000人（運用開始3ヶ月） |
| **フォーマット** | 30秒〜1分のショートアニメ |
| **プラットフォーム** | YouTube Shorts |
| **投稿頻度** | 週2〜3本 |
| **制作手法** | AI活用（画像生成、音声合成等） |

## キャラクター

### 1. スライム系キャラクター

| 項目 | 内容 |
|------|------|
| **テーマ** | 「日常のタイミングが悪いこと」 |
| **分類** | 共感型 |
| **コンセプト** | タイミングが絶望的に悪いスライム。めげずに頑張る姿が愛おしい |
| **決め台詞** | 「...だと思った」 |

### 2. ブーメランキャラクター

| 項目 | 内容 |
|------|------|
| **テーマ** | 「小さな復讐が成功する」 |
| **分類** | 爽快型 |
| **コンセプト** | 因果応報を司る存在。悪意ある行動を見つけると、同じことが本人に返る |
| **決め台詞** | 「返ってきた」 |
